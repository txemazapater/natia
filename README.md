# NATIA

**NATIA** is an experimental native desktop AI workbench.

## Vision

NATIA is built around a simple idea:

> An AI agent should behave like a real desktop application.

### Principles

- Native first.
- Fast startup.
- Efficient multiprocess architecture.
- OpenAI-compatible API (without vendor lock-in).
- Local-first and self-host friendly.
- Extensible through tools, MCP and plugins.
- Open architecture and community-driven.

This repository starts as a vision document and will evolve into the reference implementation.

---

> "Native AI deserves native software."
